# Support

Thanks for using `background_tracker`!

## issues

If you encounter native crashes, logic bugs, or documentation errors:
1.  Search existing **Issues** to see if it's already reported.
2.  Open a new Issue with:
    *   **Device Info:** (e.g., Pixel 6, Android 13 or iPhone 12, iOS 16).
    *   **Logs:** Attach logs from `adb logcat` or Xcode Console.
    *   **Reproduction:** Steps to reproduce the bug.

## Feature Requests

We welcome ideas!
- Please verify if the feature is already in the **Roadmap** in `README.md`.
- Open a generic issue labeled "Enhancement".

## Contributions

1.  Fork the repo.
2.  Create a feature branch (`git checkout -b feature/amazing-feature`).
3.  Commit your changes.
4.  Push to the branch.
5.  Open a Pull Request.

## Security

If you discover a security vulnerability (e.g., leaking location data), please email privacy@example.com instead of opening a public issue.
